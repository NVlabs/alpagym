# AlpaGym decks

Published slide decks for **AlpaGym**, hosted on GitHub Pages:

**→ https://nvlabs.github.io/alpagym/**

- **dev-onboarding** — a guided tour of the architecture, with clickable figures.
- **user-onboarding** — a quick start: set up, run, and add your own policy.

This is a docs-only branch. [`.github/workflows/pages.yml`](.github/workflows/pages.yml)
renders every `decks/*/slides.tpl.md` to HTML and deploys on each push. To build
or edit a deck locally, see [`decks/README.md`](decks/README.md).
